void av_force_cpu_flags ( int  flags)

Disables cpu detection and forces the specified flags. -1 is a special case that disables forcing of specific flags.

int av_get_cpu_flags ( void  )

Return the flags which specify extensions supported by the CPU. The returned value is affected by av_force_cpu_flags() if that was used before. So av_get_cpu_flags() can easily be used in a application to detect the enabled cpu flags.
