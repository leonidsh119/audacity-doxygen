int Px_GetNumMixers ( PxMixer *  mixer)

Px_GetNumMixers returns the number of mixers which could be used with the given PortAudio device.

On most systems, there will be only one mixer for each device; however there may be multiple mixers for each device, or possibly multiple mixers which are independent of any particular PortAudio device.
