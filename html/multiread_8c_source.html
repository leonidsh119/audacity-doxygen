32 /* multiread_fetch - read samples into multiple channels. */
33 /*
34  * The susp is shared by all channels. The susp has backpointers
35  * to the tail-most snd_list node of each channels, and it is by
36  * extending the list at these nodes that sounds are read in.
37  * To avoid a circularity, the reference counts on snd_list nodes
38  * do not include the backpointers from this susp. When a snd_list
39  * node refcount goes to zero, the multiread susp's free routine
40  * is called. This must scan the backpointers to find the node that
41  * has a zero refcount (the free routine is called before the node
42  * is deallocated, so this is safe). The backpointer is then set
43  * to NULL. When all backpointers are NULL, the susp itself is
44  * deallocated, because it can only be referenced through the
45  * snd_list nodes to which there are backpointers.
46  */
