252  word * xM, /* [0..12] IN */
253  word * xMc, /* [0..12] OUT */

255  word * expon_out, /* OUT */

264

266  */
267

293

304  * can be calculated by using the expononent and the mantissa part of

306  * So, this method avoids any division and uses only a scaling
