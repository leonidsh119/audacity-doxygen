1 // pvshell.c -- This is a skeleton for a Nyquist primitive that
2 // returns a sound. The sound is obtained by calling a function
3 // with a request consisting of a location to put samples and
4 // a count of how many samples are needed. The function returns
5 // the actual number of samples computed and flags indicating
6 // if the signal has reached the logical stop or termination.
7 // In addition, there are interfaces for extracting samples
8 // from input sounds.
9 // This code is designed for a time-stretching phase vocoder,
10 // but could be used for other purposes. It is derived from
11 // compose.c, which might have been implmented with this
12 // skeleton had we started out with this abstraction.
