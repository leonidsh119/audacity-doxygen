26 /* The snd_fft function is based on snd_fetch_array */
27 /*
28  * storage layout: the extra field points to extra state that we'll use
29  * extra[0] -> length of extra storage
30  * extra[1] -> CNT (number of samples in current block)
31  * extra[2] -> INDEX (current sample index in current block)
32  * extra[3] -> FILLCNT (how many samples in buffer)
33  * extra[4] -> TERMCNT (how many samples until termination)
34  * extra[5 .. 5+len-1] -> samples (stored as floats)
35  * extra[5+len .. 5+2*len-1] -> array of samples to fft
36  * extra[5+2*len ... 5+3*len-1] -> window coefficients
37  *
38  * Termination details:
39  * Return NIL when the sound terminates.
40  * Termination is defined as the point where all original
41  * signal samples have been shifted out of the samples buffer
42  * so that all that's left are zeros from beyond the termination
43  * point.
44  * Implementation: when termination is discovered, set TERMCNT
45  * to the number of samples to be shifted out. TERMCNT is initially
46  * -1 as a flag that we haven't seen the termination yet.
47  * Each time samples are shifted, decrement TERMCNT by the shift amount.
48  * When TERMCNT goes to zero, return NULL.
49  */
50
