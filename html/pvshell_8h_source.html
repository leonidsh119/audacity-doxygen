10 /* this function is called to compute samples. It should compute n
11  * samples (floats == sample_type) and store them at out[i].
12  * You can return less than n samples by writing the actual number
13  * of samples computed into *n. Normally, you return zero.
14  * To indicate that the time of the FIRST sample is the logical stop
15  * time, return PVSHELL_FLAG_LOGICAL_STOP. (If the logical stop time
16  * is not at the first sample, but instead at sample j, then just
17  * return j samples (from 0 to j-1), save the rest of the samples,
18  * and the next time, the first sample will correspond to the logical
19  * stop time, so you can return PVSHELL_FLAG_LOGICAL_STOP.
20  * To indicate that the sound has terminated, return
21  * PVSHELL_FLAG_TERMINATE. This should be the only time you return
22  * zero samples. (As with logical stop time, if you have samples to
23  * return before termination, then do it, and return
24  * PVSHELL_FLAG_TERMINATE the next time you are called, at which
25  * point you've returned all the samples, so you can set *n = 0.
26  */

51 /* to get samples from f or g, use these macros. For each sample, call
52  * PVSHELL_TEST_X to get logical stop and terminate flags (but do not
53  * fetch a sample). Then, if you want, call PVSHELL_FETCH_X to get the
54  * next sample. You can call PVSHELL_TEST_X multiple times before
55  * calling PVSHELL_FETCH_X, e.g. you can return exit a loop when you
56  * see a logical stop flag and later call PVSHELL_TEST_X again. You
57  * CANNOT call PVSHELL_FETCH_X multiples times without an intervening
58  * call to PVSHELL_TEST_X. Finally, the logical stop flag is only
59  * returned once. Normally you should write something like:
60  * new_flags = PVSHELL_TEST_F(susp);
61  * susp->flags | = new_flags; // remember flags
62  * if (new_flags) break;
63  * in the sample loop so that you will break when you see logical_stop.
64  * Outside the loop, you can return (*n ? 0 : susp->flags) which will
65  * return 0 if you computed samples before the logical stop was detected.
66  * Then the next time you are called, you will return the logical_stop
67  * flag because you saved it in susp->flags, and the flag applies to the
68  * *beginning* of the sample block. This code handles terminate too.
69  */
