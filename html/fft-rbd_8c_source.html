18 /* snd_fetch_array -- fetch a lisp array of samples */
19 /*
20  * storage layout: the extra field points to extra state that we'll use
21  * extra[0] -> length of extra storage
22  * extra[1] -> CNT (number of samples in current block)
23  * extra[2] -> INDEX (current sample index in current block)
24  * extra[3] -> FILLCNT (how many samples in buffer)
25  * extra[4] -> TERMCNT (how many samples until termination)
26  * extra[4 .. 4+len-1] -> samples (stored as floats)
27  *
28  * Termination details:
29  * Return NIL when the sound terminates.
30  * Termination is defined as the point where all original
31  * signal samples have been shifted out of the samples buffer
32  * so that all that's left are zeros from beyond the termination
33  * point.
34  * Implementation: when termination is discovered, set TERMCNT
35  * to the number of samples to be shifted out. TERMCNT is initially
36  * -1 as a flag that we haven't seen the termination yet.
37  * Each time samples are shifted, decrement TERMCNT by the shift amount.
38  * When TERMCNT goes to zero, return NULL.
39  */
