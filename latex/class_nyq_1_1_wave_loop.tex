Open the specified file and load its data. 

Data from a previously opened file will be overwritten by this function. An \hyperlink{class_nyq_1_1_stk_error}{Stk\+Error} will be thrown if the file is not found, its format is unknown, or a read error occurs. If the file data is to be loaded incrementally from disk and normalization is specified, a scaling will be applied with respect to fixed-\/point limits. If the data format is floating-\/point, no scaling is performed.
