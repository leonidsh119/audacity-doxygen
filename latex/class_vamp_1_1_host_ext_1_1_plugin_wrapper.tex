\subsection{Detailed Description}
\hyperlink{class_vamp_1_1_host_ext_1_1_plugin_wrapper}{Plugin\+Wrapper} is a simple base class for adapter plugins. It takes a pointer to a \char`\"{}to be wrapped\char`\"{} \hyperlink{namespace_vamp}{Vamp} plugin on construction, and provides implementations of all the \hyperlink{namespace_vamp}{Vamp} plugin methods that simply delegate through to the wrapped plugin. A subclass can therefore override only the methods that are meaningful for the particular adapter.

\begin{DoxyNote}{Note}
This class was introduced in version 1.\+1 of the \hyperlink{namespace_vamp}{Vamp} plugin S\+DK. 
\end{DoxyNote}
