\subsection{Detailed Description}
Portaudio W\+D\+M-\/\+KS host A\+PI. 

\begin{DoxyNote}{Note}
This is the implementation of the Portaudio host A\+PI using the Windows W\+D\+M/\+Kernel Streaming A\+PI in order to enable very low latency playback and recording on all modern Windows platforms (e.\+g. 2K, XP, Vista, Win7) Note\+: This A\+PI accesses the device drivers below the usual K\+M\+I\+X\+ER component which is normally used to enable multi-\/client mixing and format conversion. That means that it will lock out all other users of a device for the duration of active stream using those devices 
\end{DoxyNote}
