\subsection{Detailed Description}
A \hyperlink{class_prefs_panel}{Prefs\+Panel} that configures dynamic loading of \hyperlink{class_theme}{Theme} icons and colours. 

Provides\+:
\begin{DoxyItemize}
\item Button to save current theme as a single png image.
\item Button to load theme from a single png image.
\item Button to save current theme to multiple png images.
\item Button to load theme from multiple png images.
\item (Optional) Button to save theme as Cee data.
\item Button to read theme from default values in program.
\item Check\+Box for loading custom themes at startup.
\end{DoxyItemize}
