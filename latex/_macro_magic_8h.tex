\subsection{Detailed Description}
This file allows the same macros to do multiple duty by undefining and redefining the macros.

For example D\+E\+F\+I\+N\+E\+\_\+\+I\+M\+A\+GE will generate\+:


\begin{DoxyItemize}
\item extern int name;
\item int name = -\/1;
\item Register\+Image( name, initiialiser, textual\+\_\+name);
\end{DoxyItemize}
