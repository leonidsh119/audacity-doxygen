\subsection{Detailed Description}
Given a string representing a command, pass it to the \hyperlink{class_batch_commands}{Batch\+Commands} system. 

The eventual aim is to move the code from \hyperlink{class_batch_commands}{Batch\+Commands} out into separate command classes, but until this happens, \hyperlink{class_batch_eval_command}{Batch\+Eval\+Command} can act as a \textquotesingle{}bridge\textquotesingle{} to that system.
