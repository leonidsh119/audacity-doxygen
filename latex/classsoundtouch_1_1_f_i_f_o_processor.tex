\subsection{Detailed Description}
Base-\/class for sound processing routines working in F\+I\+FO principle. With this base class it\textquotesingle{}s easy to implement sound processing stages that can be chained together, so that samples that are fed into beginning of the pipe automatically go through all the processing stages.

When samples are input to this class, they\textquotesingle{}re first processed and then put to the F\+I\+FO pipe that\textquotesingle{}s defined as output of this class. This output pipe can be either other processing stage or a F\+I\+FO sample buffer.
