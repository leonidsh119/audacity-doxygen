\subsection{Detailed Description}
Moves data from one place to another, converting it as required. 

\hyperlink{class_shuttle}{Shuttle} provides a base class for transfering parameter data into and out of clasess into some other structure. This is a common requirement and is needed for\+:
\begin{DoxyItemize}
\item Prefs data
\item \hyperlink{class_command}{Command} line parameter data
\item Project data in X\+ML
\end{DoxyItemize}

The \textquotesingle{}Master\textquotesingle{} is the string side of the shuttle transfer, the \textquotesingle{}Client\textquotesingle{} is the binary data side of the transfer.
