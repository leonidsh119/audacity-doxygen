This function can be used to perform the initialization of wx\+Widgets if you can\textquotesingle{}t use the default initialization code for any reason.

If the function returns true, the initialization was successful and the global \hyperlink{classwx_app}{wx\+App} object \hyperlink{app_8h_a01fd2cb0a8fbaade87800e71a50e855d}{wx\+The\+App} has been created. Moreover, \hyperlink{group__group__funcmacro__appinitterm_gaca1c6e45c8241bb2b16bee8f09401bc8}{wx\+Entry\+Cleanup()} must be called afterwards. If the function returns false, a catastrophic initialization error occurred and (at least the G\+UI part of) the library can\textquotesingle{}t be used at all.

Notice that parameters {\ttfamily argc} and {\ttfamily argv} may be modified by this function.
