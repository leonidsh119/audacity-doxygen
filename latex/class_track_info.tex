\subsection{Detailed Description}
The \hyperlink{class_track_info}{Track\+Info} is shown to the side of a track It has the menus, pan and gain controls displayed in it. So \char`\"{}\+Info\char`\"{} is somewhat a misnomer. Should possibly be \char`\"{}\+Track\+Controls\char`\"{}. 

\hyperlink{class_track_panel}{Track\+Panel} and not \hyperlink{class_track_info}{Track\+Info} takes care of the functionality for each of the buttons in that panel.

In its current implementation \hyperlink{class_track_info}{Track\+Info} is not derived from a wx\+Window. Following the original coding style, it has been coded as a \textquotesingle{}flyweight\textquotesingle{} class, which is passed state as needed, except for the array of gains and pans.

If we\textquotesingle{}d instead coded it as a wx\+Window, we would have an instance of this class for each instance displayed.
