\subsection{Detailed Description}
Based on \hyperlink{class_theme_base}{Theme\+Base}, \hyperlink{class_theme}{Theme} manages image and icon resources. 

\hyperlink{class_theme}{Theme} is a class which manages theme resources. It maps sets of ids to the resources and to names of the resources, so that they can be loaded/saved from files.

\hyperlink{class_theme}{Theme} adds the Audacity specific images to \hyperlink{class_theme_base}{Theme\+Base}.
