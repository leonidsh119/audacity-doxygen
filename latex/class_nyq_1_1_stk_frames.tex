\subsection{Detailed Description}
An S\+TK class to handle vectorized audio data. 

This class can hold single-\/ or multi-\/channel audio data in either interleaved or non-\/interleaved formats. The data type is always Stk\+Float. In an effort to maintain efficiency, no out-\/of-\/bounds checks are performed in this class.

Possible future improvements in this class could include functions to inter-\/ or de-\/interleave the data and to convert to and return other data types.

by Perry R. Cook and Gary P. Scavone, 1995 -\/ 2005.
