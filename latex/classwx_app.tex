\subsection{Detailed Description}
The \hyperlink{classwx_app}{wx\+App} class represents the application itself when {\ttfamily wx\+U\+S\+E\+\_\+\+G\+UI=1}.

In addition to the features provided by \hyperlink{classwx_app_console}{wx\+App\+Console} it keeps track of the {\itshape top window} (see \hyperlink{classwx_app_a39033ec4c79bc1871c91ada7b31941e6}{Set\+Top\+Window()}) and adds support for video modes (see Set\+Video\+Mode()).

In general, application-\/wide settings for G\+U\+I-\/only apps are accessible from \hyperlink{classwx_app}{wx\+App} (or from wx\+System\+Settings or wx\+System\+Options classes).
