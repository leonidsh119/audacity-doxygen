\subsection{Detailed Description}
\hyperlink{class_vamp_1_1_plugin_adapter}{Plugin\+Adapter} and \hyperlink{class_vamp_1_1_plugin_adapter_base}{Plugin\+Adapter\+Base} provide a wrapper class that a plugin library can use to make its C++ \hyperlink{class_vamp_1_1_plugin}{Vamp\+::\+Plugin} objects available through the \hyperlink{namespace_vamp}{Vamp} C A\+PI.

Almost all \hyperlink{namespace_vamp}{Vamp} plugin libraries will want to make use of this. To do so, all they need to do is declare a Plugin\+Adapter$<$\+T$>$ for each plugin class T in their library. It\textquotesingle{}s very simple, and you need to know absolutely nothing about how it works in order to use it. Just cut and paste from an existing plugin\textquotesingle{}s discovery function. \begin{DoxySeeAlso}{See also}
