\subsection{Detailed Description}
A type of factory for Commands of various sorts. 

\hyperlink{class_command_builder}{Command\+Builder} has the task of deciding what command is meant by a given command string, and producing a suitable command object. For now, it doesn\textquotesingle{}t actually do any processing -\/ it just passes everything on to the Batch\+Command system by constructing Batch\+Command\+Eval objects.
