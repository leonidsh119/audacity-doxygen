Read sample frames from the file into an \hyperlink{class_nyq_1_1_stk_frames}{Stk\+Frames} object. 

The number of sample frames to read will be determined from the number of frames of the \hyperlink{class_nyq_1_1_stk_frames}{Stk\+Frames} argument. If this size is larger than the available data in the file (given the file size and starting frame index), the extra frames will be unaffected (the \hyperlink{class_nyq_1_1_stk_frames}{Stk\+Frames} object will not be resized). Optional parameters are provided to specify the starting sample frame within the file (default = 0) and whether to normalize the data with respect to fixed-\/point limits (default = true). An \hyperlink{class_nyq_1_1_stk_error}{Stk\+Error} will be thrown if a file error occurs or if the number of channels in the \hyperlink{class_nyq_1_1_stk_frames}{Stk\+Frames} argument is not equal to that in the file.
