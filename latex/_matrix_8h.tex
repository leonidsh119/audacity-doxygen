\subsection{Detailed Description}
General routine to interpolate (or even extrapolate small amounts) audio when a few of the samples are bad. Works great for a few dozen bad samples, but not so well with hundreds. Uses the least-\/squares autoregression (L\+S\+AR) algorithm, as described in\+: 

Holds both the \hyperlink{class_matrix}{Matrix} and \hyperlink{class_vector}{Vector} classes, supporting linear algebra operations, including matrix inversion. Used by Interpolate\+Audio.

Simon Godsill, Peter Rayner, and Olivier Cappe. Digital Audio Restoration. Berlin\+: Springer, 1998.

This is the same work used by Gnome Wave Cleaner (G\+WC), however this implementation is original.
