\subsection{Detailed Description}
Internationalisation support. 

This class is used to help internationalisation and in general compatibility with different locales and character sets. It deals mostly with converting numbers, but also has important functions to convert to/from U\+T\+F-\/8, which is used in X\+ML files and on Mac OS X for the filesystem.
