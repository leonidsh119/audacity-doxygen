\subsection{Detailed Description}
S\+TK envelope base class. 

Draggable curve used in \hyperlink{class_track_panel}{Track\+Panel} for varying amplification.

This class implements a simple envelope generator which is capable of ramping to a target value by a specified {\itshape rate}. It also responds to simple {\itshape key\+On} and {\itshape key\+Off} messages, ramping to 1.\+0 on key\+On and to 0.\+0 on key\+Off.

by Perry R. Cook and Gary P. Scavone, 1995 -\/ 2005.

This class manages an envelope -\/ i.\+e. a piecewise linear funtion that the user can edit by dragging control points around. The envelope is most commonly used to control the amplitude of a waveform, but it is also used to shape the Equalization curve.
