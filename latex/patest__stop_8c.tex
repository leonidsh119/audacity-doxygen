\subsection{Detailed Description}
\hyperlink{struct_test}{Test} different ways of stopping audio. 

\hyperlink{struct_test}{Test} the three ways of stopping audio\+:
\begin{DoxyItemize}
\item calling \hyperlink{portaudio_8h_af18dd60220251286c337631a855e38a0}{Pa\+\_\+\+Stop\+Stream()},
\item calling \hyperlink{portaudio_8h_a138e57abde4e833c457b64895f638a25}{Pa\+\_\+\+Abort\+Stream()},
\item and returning a 1 from the callback function.
\end{DoxyItemize}

A long latency is set up so that you can hear the difference. Then a simple 8 note sequence is repeated twice. The program will print what you should hear.
