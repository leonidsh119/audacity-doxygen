\subsection{Detailed Description}
This file contains a general function which will import almost any type of sampled audio file (i.\+e. anything except M\+I\+DI) and return the tracks that were imported. This function just figures out which one to call; the actual importers are in Import\+P\+CM, Import\+M\+P3, Import\+O\+GG, Import\+Raw\+Data, Import\+L\+OF, Import\+QT and Import\+F\+L\+AC.
