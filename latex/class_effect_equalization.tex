\subsection{Detailed Description}
An \hyperlink{class_effect}{Effect} that modifies volume in different frequency bands. 

Performs filtering, using an F\+FT to do a F\+IR filter. It lets the user draw an arbitrary envelope (using the same envelope editing code that is used to edit the track\textquotesingle{}s amplitude envelope).

Also allows the curve to be specified with a series of \textquotesingle{}graphic EQ\textquotesingle{} sliders.

The filter is applied using overlap/add of Hanning windows.

Clone of the F\+FT \hyperlink{class_filter}{Filter} effect, no longer part of Audacity.
