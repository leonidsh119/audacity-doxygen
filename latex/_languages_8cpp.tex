\subsection{Detailed Description}
Determine installed languages. 

Figure out what translations are installed and return a list of language codes (like \char`\"{}es\char`\"{}, \char`\"{}fr\char`\"{}, or \char`\"{}pt-\/br\char`\"{}) and corresponding language names (like \char`\"{}\+Español\char`\"{}, \char`\"{}\+Français\char`\"{}, and \char`\"{}\+Português\char`\"{}). We use our own list of translations of language names (i.\+e. \char`\"{}\+Français\char`\"{} instead of \char`\"{}\+French\char`\"{}) but we fallback on the language name in wx\+Widgets if we don\textquotesingle{}t have it listed.

This code is designed to work well with all of the current languages, but adapt to any language that wx\+Widgets supports. Other languages will only be supported if they\textquotesingle{}re added to the database using wx\+Locale\+::\+Add\+Language.

But for the most part, this means that somebody could add a N\+EW translation and have it work immediately.
