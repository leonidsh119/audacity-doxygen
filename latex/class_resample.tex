\subsection{Detailed Description}
Interface to libsoxr. 

This class abstracts the interface to different resampling libraries\+:

libsoxr, written by Rob Sykes. L\+G\+PL.

Since Audacity always does resampling on mono streams that are contiguous in memory, this class doesn\textquotesingle{}t support multiple channels or some of the other optional features of some of these resamplers.

Main processing function. Resamples from the input buffer to the output buffer. 

Reads samples from the input buffer, and writes samples to the output buffer. Stops when either is exhaughsted, or we reach a convenient block end, unless last\+Flag is set to force emptying the input buffer. The number of input samples used is returned in in\+Buffer\+Used, and the number of output samples generated is the return value of the function. This function may do nothing if you don\textquotesingle{}t pass a large enough output buffer (i.\+e. there is no where to put a full block of output data) 
\begin{DoxyParams}{Parameters}
{\em factor} & The scaling factor to resample by. \\
\hline
{\em in\+Buffer} & Buffer of input samples to be processed (mono) \\
\hline
{\em in\+Buffer\+Len} & Length of the input buffer, in samples. \\
\hline
{\em last\+Flag} & Flag to indicate this is the last lot of input samples and the buffer needs to be emptied out into the rate converter. \\
\hline
{\em in\+Buffer\+Used} & Number of samples from in\+Buffer that have been used (unless last\+Flag is true, we don\textquotesingle{}t garuntee to process all the samples in the input this time, we may leave some for next time) \\
\hline
{\em out\+Buffer} & Buffer to write output (converted) samples to. \\
\hline
{\em out\+Buffer\+Len} & How big out\+Buffer is. \\
