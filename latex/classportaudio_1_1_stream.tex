\subsection{Detailed Description}
A \hyperlink{classportaudio_1_1_stream}{Stream} represents an active or inactive input and/or output data stream in the \hyperlink{classportaudio_1_1_system}{System}. 

Concrete \hyperlink{classportaudio_1_1_stream}{Stream} classes should ensure themselves being in a closed state at destruction (i.\+e. by calling their own \hyperlink{classportaudio_1_1_stream_acb79f85f43b9f87b1e3c495d5f02ae58}{close()} method in their deconstructor). Following good C++ programming practices, care must be taken to ensure no exceptions are thrown by the deconstructor of these classes. As a consequence, clients need to explicitly call \hyperlink{classportaudio_1_1_stream_acb79f85f43b9f87b1e3c495d5f02ae58}{close()} to ensure the stream closed successfully.

The \hyperlink{classportaudio_1_1_stream}{Stream} object can be used to manipulate the \hyperlink{classportaudio_1_1_stream}{Stream}\textquotesingle{}s state. Also, time-\/constant and time-\/varying information about the \hyperlink{classportaudio_1_1_stream}{Stream} can be retreived.
