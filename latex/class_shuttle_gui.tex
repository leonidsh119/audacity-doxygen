\subsection{Detailed Description}
Derived from \hyperlink{class_shuttle_gui_base}{Shuttle\+Gui\+Base}, an Audacity specific class for shuttling data to and from G\+UI. 

\hyperlink{class_shuttle_gui}{Shuttle\+Gui} extends the idea of the data \hyperlink{class_shuttle}{Shuttle} class to include creation of dialog controls. As part of this it provides an interface to sizers that leads to shorter more readable code.

It also allows the code that is used to create dialogs to be reused to shuttle information in and out.

Most of the \hyperlink{class_shuttle_gui}{Shuttle\+Gui} functions are actually defined in \hyperlink{class_shuttle_gui_base}{Shuttle\+Gui\+Base}.
\begin{DoxyItemize}
\item wx\+Widgets widgets are dealt with by \hyperlink{class_shuttle_gui_base}{Shuttle\+Gui\+Base}.
\item Audacity specific widgets are dealt with by \hyperlink{class_shuttle_gui}{Shuttle\+Gui}
\end{DoxyItemize}
