Setup the picture fields based on the specified image parameters and the provided image data buffer.

The picture fields are filled in by using the image data buffer pointed to by ptr.

If ptr is N\+U\+LL, the function will fill only the picture linesize array and return the required size for the image buffer.

To allocate an image buffer and fill the picture data in one call, use \hyperlink{group__lavc__picture_ga03b764a93c34d00e5a33a5ebb0b4a81b}{avpicture\+\_\+alloc()}.
