\subsection{Detailed Description}
All \hyperlink{class_audacity_project}{Audacity\+Project} functions that provide the menus. Implements \hyperlink{class_audacity_project_command_functor}{Audacity\+Project\+Command\+Functor}. 

This file implements the method that creates the menu bar, plus all of the methods that get called when you select an item from a menu.

All of the menu bar handling is part of the class \hyperlink{class_audacity_project}{Audacity\+Project}, but the event handlers for all of the menu items have been moved to \hyperlink{_menus_8h}{Menus.\+h} and \hyperlink{_menus_8cpp}{Menus.\+cpp} for clarity.
