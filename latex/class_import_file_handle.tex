\subsection{Detailed Description}
An \hyperlink{class_import_file_handle}{Import\+File\+Handle} for data. 

Base class for Flac\+Import\+File\+Handle, \hyperlink{class_l_o_f_import_file_handle}{L\+O\+F\+Import\+File\+Handle}, \hyperlink{class_m_p3_import_file_handle}{M\+P3\+Import\+File\+Handle}, Ogg\+Import\+File\+Handle and \hyperlink{class_p_c_m_import_file_handle}{P\+C\+M\+Import\+File\+Handle}. Gives A\+PI for sound file import.

The Ogg format supports multiple logical bitstreams that can be chained within the physical bitstream. The sampling rate and number of channels can vary between these logical bitstreams. For the moment, we\textquotesingle{}ll ignore all but the first logical bitstream.

Ogg also allows for an arbitrary number of channels. Luckily, so does Audacity. We\textquotesingle{}ll call the first channel Left\+Channel, the second Right\+Channel, and all others after it Mono\+Channel.
