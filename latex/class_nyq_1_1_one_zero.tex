Set the zero position in the z-\/plane. 

This method sets the zero position along the real-\/axis of the z-\/plane and normalizes the coefficients for a maximum gain of one. A positive zero value produces a high-\/pass filter, while a negative zero value produces a low-\/pass filter. This method does not affect the filter {\itshape gain} value.
