\subsection{Detailed Description}
Base class which encapsulates a process. 

That process may depend on certain parameters (determined by the command\textquotesingle{}s signature) and may produce output on various channels. Any process which is to be controlled by a script should be separated out into its own \hyperlink{class_command}{Command} class. (And that class should be registered with the \hyperlink{class_command_directory}{Command\+Directory}).
