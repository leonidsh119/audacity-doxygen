Prototype for plugin accessor function.

Plugins are discovered by hosts using R\+DF data (not by loading libraries). See \href{http://lv2plug.in}{\tt http\+://lv2plug.\+in} for details on the discovery process, though most hosts should use an existing library to implement this functionality.

This is the simple plugin discovery A\+PI, suitable for most statically defined plugins. Advanced plugins that need access to their bundle during discovery can use \hyperlink{lv2_8h_a9eb5804c670849d3faafed9f9cd65f23}{lv2\+\_\+lib\+\_\+descriptor()} instead. Plugin libraries M\+U\+ST include a function called \char`\"{}lv2\+\_\+descriptor\char`\"{} or \char`\"{}lv2\+\_\+lib\+\_\+descriptor\char`\"{} with C-\/style linkage, but S\+H\+O\+U\+LD provide \char`\"{}lv2\+\_\+descriptor\char`\"{} wherever possible.

When it is time to load a plugin (designated by its U\+RI), the host loads the plugin\textquotesingle{}s library, gets the \hyperlink{bindings__test__plugin_8c_affedc7aabe2715522365f21a9c1ff65d}{lv2\+\_\+descriptor()} function from it, and uses this function to find the L\+V2\+\_\+\+Descriptor for the desired plugin. Plugins are accessed by index using values from 0 upwards. This function M\+U\+ST return N\+U\+LL for out of range indices, so the host can enumerate plugins by increasing {\ttfamily index} until N\+U\+LL is returned.

Note that {\ttfamily index} has no meaning, hosts M\+U\+ST N\+OT depend on it remaining consistent between loads of the plugin library.

The \hyperlink{bindings__test__plugin_8c_affedc7aabe2715522365f21a9c1ff65d}{lv2\+\_\+descriptor()} function is the entry point to the plugin library. The host will load the library and call this function repeatedly with increasing indices to find all the plugins defined in the library. The index is not an indentifier, the U\+RI of the returned descriptor is used to determine the identify of the plugin.

This method is in the ``discovery\textquotesingle{}\textquotesingle{} threading class, so no other functions or methods in this plugin library will be called concurrently with it.
