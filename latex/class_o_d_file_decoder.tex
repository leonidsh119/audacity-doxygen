Decodes the samples for this blockfile from the real file into a float buffer. This is file specific, so subclasses must implement this only. the buffer should be created by the \hyperlink{class_o_d_file_decoder}{O\+D\+File\+Decoder} implementing this method. It should set the format parameter so that the client code can deal with it. This class should call \hyperlink{class_o_d_file_decoder_a0aa7f2105d4a4f80871987b2ef2a484f}{Read\+Header()} first, so it knows the length, and can prepare the file object if it needs to. returns negative value for failure, 0 or positive value for success. 

Implemented in \hyperlink{class_o_d_flac_decoder_a5a69610efd606948ab6fad4de6489d87}{O\+D\+Flac\+Decoder}.
